Контрольні питання

1. Що таке HTML?

HTML — це мова розмітки, яку веб-розробники використовують для структурування та описати вміст веб-сторінки

2. Яким чином забезпечується форматування тексту в HTML-документі?

За допомогою стилювання css, вибираючи тегом бокс з текстом

3. Яка мета розробки HTML5?

Поліпшення рівня підтримки мультимедіатехнологій з одночасним збереженням зворотної сумісності, зручності читання коду для людини і простоти аналізу для парсерів

4. Який синтаксис HTML?

<назва_тегу> контент </назва_тегу>

5. Структура HTML-документа?

<!DOCTYPE html>

<html>

<head>

<title> Назва сторінки</title>

</head>

<body>

<h1>Заголовок статті</h1>

<p>Абзац тексту</p>

</body>

</html>

6. Яким чином здійснюється взаємодія між HTML-документом і сервером?

HTML документи можуть бути транспортовані так само як і будь-які інші файли (наприклад, за допомогою протоколів FTP, TCP), проте зазвичай вони транспортуються із сервера за допомогою протоколу HTTP

7. Які текстові редактори для роботи з HTML Ви знаєте?

VS code блокнот

8. Що означає DOCTYPE у HTML-документі?

призначений для вказівки типу поточного документа

9. Що таке заголовок HTML-документа?

Тег header, контент, що не рендеритимися, невидимий на сторінці. Як ім'я в закладці, підключення шрифтів й бібліотек

10. Що таке тіло HTML-документа?
